# Screenshots for documentation / final report

Add **PNG** or **JPG** files here and reference them from your report or README.

## How to capture

1. **Main UI:** Open local `http://localhost:5173` or your Vercel URL; show the empty form (resume + job description). Save as `01-home.png`.  
2. **Results:** After **Analyze match**, capture the score and tips. Save as `02-results.png`.  
3. **API docs:** Open `https://YOUR-RENDER-URL.onrender.com/docs`; show `POST /api/analyze`. Save as `03-swagger.png`.  
4. **Deployed app (optional):** Full-page shot of the live Vercel site. Save as `04-vercel-live.png`.
